# State Street Financial Performance Analysis

## Objective
To analyze State Street’s financial performance using real annual report data and present insights through an executive Power BI dashboard.

## Data Source
Publicly available State Street annual reports (2019–2021).

## Tools Used
- Microsoft Excel
- Power BI

## Key Analysis
- Revenue and profitability trend analysis
- Profit margin and growth evaluation
- Asset growth analysis (AUM and AUC/A)
- COVID impact and post-pandemic recovery

## Outcome
Built an interactive Power BI dashboard to support data-driven financial insights and executive decision-making.
